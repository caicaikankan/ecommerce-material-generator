# 电商商品素材自动生成工具

一个帮助电商运营人员快速生成商品主图草稿和配套文案的在线工具。

## 🌟 功能特点

- **商品信息录入**：简洁的表单界面，快速录入商品基本信息
- **素材自动生成**：基于录入信息自动生成商品主图草稿
- **配套文案生成**：智能生成商品描述和营销文案
- **现代化界面**：采用Tailwind CSS构建的响应式设计
- **即开即用**：纯HTML实现，无需安装，直接在浏览器中打开使用

## 🚀 使用方法

1. 下载项目文件到本地
2. 直接用浏览器打开 `电商v2.html` 文件
3. 按照页面提示录入商品信息
4. 点击生成按钮获取素材和文案

## 🛠️ 技术栈

- **前端框架**：Tailwind CSS
- **图标库**：Font Awesome
- **设计语言**：现代化渐变背景，玻璃拟态效果
- **兼容性**：支持现代浏览器
